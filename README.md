# 📊 نظام إدارة المبيعات - غزارة للتجارة والتسويق (Ghazara Sales Manager)

منصة رقمية متكاملة ومتقدمة مصممة خصيصاً لشركة **غزارة للتجارة والتسويق**، تهدف إلى أتمتة عمليات إدارة المبيعات، تتبع أداء الفريق، وتحليل البيانات باستخدام تقنيات الذكاء الاصطناعي لتحقيق أقصى درجات الكفاءة والنمو.


## 🚀 نظرة عامة (Analysis)

في بيئة عمل ديناميكية مثل قطاع التجارة والتسويق، يصبح تتبع المبيعات والعمولات يدوياً عائقاً أمام النمو. يأتي هذا النظام ليحل هذه التحديات من خلال توفير لوحة تحكم مركزية تربط بين الموظفين، المبيعات، والأهداف الاستراتيجية للشركة.

### **لماذا ينجح هذا النظام؟**
- **المركزيّة:** يجمع كافة بيانات المبيعات والعمولات في مكان واحد.
- **الذكاء الاصطناعي:** يوفر مساعداً ذكياً لتحليل التقارير وتقديم توصيات استراتيجية.
- **الشفافية:** يتيح لكل موظف معرفة مستوى إنجازه ونسبة تحقيق أهدافه لحظة بلحظة.
- **القابلية للتوسع:** مبني على بنية تحتية حديثة تسمح بإضافة مميزات جديدة بسهولة.


## ✨ المميزات الرئيسية (Solutions)

| الميزة | الوصف | الفائدة الاستراتيجية |
| :--- | :--- | :--- |
| **لوحة تحكم ذكية** | عرض فوري لإجمالي الإيرادات، المبيعات، ونسب الإنجاز. | اتخاذ قرارات سريعة بناءً على بيانات حقيقية. |
| **إدارة فريق المبيعات** | ملفات تعريفية للموظفين، تتبع الرتب، والعمولات. | تحسين إدارة الموارد البشرية وتقدير الأداء. |
| **تتبع الأهداف (Targets)** | وضع أهداف شهرية/ربع سنوية ومتابعة نسبة التحقيق. | ضمان توافق أداء الفريق مع رؤية الشركة. |
| **مساعد التقارير (AI)** | دردشة تفاعلية مدعومة بالذكاء الاصطناعي لتحليل البيانات. | الحصول على إحصائيات معقدة بلغة طبيعية بسيطة. |
| **إدارة الفئات** | تصنيف الإيرادات حسب نوع النشاط أو الخدمة. | معرفة أكثر القطاعات ربحية وتوجيه الجهود إليها. |


## 🛠 البناء التقني (Tech Stack)

تم اختيار الأدوات بعناية لضمان السرعة، الأمان، وسهولة الصيانة:

- **Frontend:** [React.js](https://reactjs.org/) مع [Vite](https://vitejs.dev/) لتجربة تطوير فائقة السرعة.
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) مع مكونات [Radix UI](https://www.radix-ui.com/) و [Shadcn/UI].
- **Backend & SDK:** مدعوم بواسطة [Base44](https://base44.com/) لإدارة الكيانات، الهوية، والذكاء الاصطناعي.
- **Charts:** [Recharts](https://recharts.org/) لتمثيل البيانات بصرياً.
- **Animations:** [Framer Motion](https://www.framer.com/motion/) لتجربة مستخدم سلسة.
- **State Management:** [TanStack Query](https://tanstack.com/query/latest) لإدارة جلب البيانات وتزامنها.


## 🏗 خطوات التنفيذ (Implementation)

### **1. المتطلبات الأولية**
- Node.js (إصدار 18 أو أحدث)
- حساب على منصة Base44 للحصول على المفاتيح البرمجية.

### **2. التثبيت**
```bash
# استنساخ المستودع
git clone https://github.com/Tomybarq/Ghazara-App-Sales-manager-.git

# الدخول إلى المجلد
cd Ghazara-App-Sales-manager-

# تثبيت التبعيات
npm install
```

### **3. التشغيل**
```bash
# تشغيل المشروع في بيئة التطوير
npm run dev
```


## 📈 نموذج العمل والقيمة المضافة (Business Model)

1. **Quick Win (فوز سريع):** أتمتة حساب العمولات وإصدار تقارير المبيعات اليومية بدلاً من الجداول اليدوية، مما يوفر 10+ ساعات عمل أسبوعياً للإدارة.
2. **الحل الاستراتيجي:** بناء قاعدة بيانات ضخمة (Big Data) لتوجهات السوق وسلوك العملاء، مما يسمح لشركة غزارة بالتنبؤ بالمبيعات المستقبلية وتعديل استراتيجياتها التسويقية.
3. **التوسع (Scalability):** يمكن تحويل هذا النظام إلى منتج SaaS (برمجيات كخدمة) لشركات التسويق الأخرى في المنطقة.
